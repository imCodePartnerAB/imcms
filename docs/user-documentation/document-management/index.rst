Document Management
===================

Basic document management is done using the *Document Manager* and *Page Information*.

You can open it through the :doc:`Admin Panel </user-documentation/admin-panel>` or in separate windows using following links:

* Document Manager - ``<domain-name>/api/admin/documents``
* Page Info - ``<domain-name>/api/admin/page-info?meta-id=<document-id>``
