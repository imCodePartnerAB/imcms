De anslag som visas i fönstrets vänstra del är rubrikerna till de senaste anslagen som skrivits i respektive avdelning.


Att skapa anslag/ svar
Ett anslag skapas genom att man klickar på "Nytt Anslag", skriver ett inlägg i inmatningsrutan och avslutar med att trycka på knappen "Skicka". Använd knappen "Ångra" för att avbryta utan att spara anslaget.
Ett svar skapas genom att man klickar på "Svara", skriver sitt inlägg i inmatningsrutan och avslutar med att trycka på knappen "Skicka". Använd knappen "Ångra" för att avbryta utan att skicka svaret.

Övrigt: Mellan rubrikfält och textfält finns en länk, "LÄNK". När man trycker på den visas en dialogruta där man anger den webadress/mailadress som skall ingå i textfältet. När man trycker på "OK" skapas automatiskt den html kod som behövs för att infoga länken. Använd knappen "Ångra" för att avbryta utan att infoga länken.
